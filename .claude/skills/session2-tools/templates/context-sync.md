# 나만의 Context Sync

이 스킬이 호출되면 아래 순서로 실행한다.

---

## 수집 범위

### Slack
- 채널: [수집할 채널 이름 — 예: #세일즈셀, #general]
- 범위: 최근 24시간
- 추출할 정보:
  - 나에 대한 멘션 (@me)
  - 중요 키워드가 포함된 메시지 (예: 긴급, 확인 필요, 결정)
  - 미답변 DM

### Notion
- 페이지: [수집할 Notion 페이지/DB 이름 — 예: 업무 일지, 미팅 노트]
- 범위: 오늘 날짜 기준
- 추출할 정보:
  - 오늘 마감 할일
  - 오늘 예정된 미팅
  - 어제 이후 업데이트된 중요 페이지

### Google Sheets
- 시트: [수집할 시트 이름 — 예: 파이프라인 현황, KPI 대시보드]
- 추출할 정보:
  - 상태별 건수 요약
  - 오늘 마감 항목
  - 어제 이후 변경된 셀

---

## 실행 흐름

subagent를 사용해 병렬로 수집한다:

```
Task 1 (Slack 수집):   → 멘션, 중요 메시지 추출
Task 2 (Notion 수집):  → 할일, 미팅 추출      } 동시 실행
Task 3 (GSheets 수집): → 현황 데이터 추출
```

수집 완료 후 하나의 브리핑 문서로 합친다.

---

## 출력 포맷

### 마크다운 형식 (기본)

```markdown
# 오늘의 컨텍스트 — [날짜]

## 📬 Slack
- [중요 메시지 1]
- [중요 메시지 2]
→ 미답변: X건

## ✅ Notion 할일
- [ ] [오늘 마감 항목 1]
- [ ] [오늘 마감 항목 2]

## 📊 파이프라인 현황
- 진행 중: X건
- 이번 주 마감: X건
- 신규 리드: X건
```

---

## 커스터마이징 포인트

1. **수집 채널/페이지 변경**: `수집 범위` 섹션에서 직접 수정
2. **추출 조건 변경**: 키워드, 날짜 범위 조정
3. **Output 형식 변경**: 마크다운 → Slack 메시지 → Notion 페이지
4. **수집 도구 추가/제거**: 사용하지 않는 섹션 삭제
