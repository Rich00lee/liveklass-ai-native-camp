• 3/10(화) 20:00~22:00 Session 4 — 고도화 + 발표 준비 [온/오프라인]

📍 오프라인 장소: 헤이그라운드 세미나실

💻 Session 1 전까지 준비해주세요
1. 노트북 지참 (Mac 권장, Windows도 가능)
2. Claude 계정은 회사에서 제공 예정이라 별도 가입 불필요합니다 — 세션 전에 초대 메일 드릴게요
3. 터미널 한번 열어보기 (Mac: Cmd+Space → "Terminal" 검색)
4. 자동화하고 싶은 업무 1가지 구체적으로 정리해오기

🎯 캠프가 끝나면
• 내 반복 업무를 자동화하는 나만의 Skill 1개 완성
• GitHub에 내 프로젝트 올리기
• 3/13(금) 전사 앞에서 데모 발표

Slack 캠프 채널에 초대해드릴게요. 궁금한 점은 언제든 DM 주세요!
```

• Session 1: 설치 오류 실시간 지원 (Node.js, npm, 터미널 환경 등)
• Session 2: MCP 설정 직접 지원 (Slack API 토큰, Notion Integration 등)
• Session 3~4: Skill 제작 시 기술 질문 서포트
• Session 5: 기술 Q&A 담당
• 캠프 이후: 팀 내 AI 활용 확산 서포터

💻 Session 1 전까지 준비해주세요
1. 노트북 지참
2. Claude 계정은 회사에서 제공 예정 — 세션 전에 초대 메일 드릴게요
3. Claude Code CLI 사전 설치 및 테스트 (npm install -g @anthropic-ai/claude-code)
4. 가능하면 MCP 서버 1개 미리 연결해보기 (Slack 또는 Notion)

💡 참고
앰버서더도 본인의 자동화 프로젝트를 동시에 진행합니다. 강의하는 역할이 아니라, 함께 배우면서 기술 부분만 서포트하는 거예요.

Slack 캠프 채널에 초대해드릴게요. 세션 전에 앰버서더 브리핑을 별도로 한번 드리겠습니다!
```
