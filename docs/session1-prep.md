## 1. 실습용 데이터

### CS 문의 메일 샘플 3개

캠프에서 데모로 보여줄 실제 업무 예시. Block 1에서 사용.

**샘플 1: 환불 문의**
```
고객: "결제한 지 5일 됐는데 환불 가능한가요? 강의가 기대와 달라서요."
→ 라이브클래스 환불 정책 기준 CS 응답 초안 작성
```

**샘플 2: 기능 문의**
```
고객: "VOD 강의를 수강생이 다운로드할 수 없게 막을 수 있나요? 보안이 중요한 기업 교육이라서요."
→ 보안 5-pack (다운로드 차단/캡처 방지/중복 로그인/워터마크/플레이어) 기반 응답
```

**샘플 3: 기술 지원**
```
고객: "카카오 소셜 로그인 연동하려는데 어떻게 설정하나요? 개발자가 없어서 직접 해야 합니다."
→ 단계별 가이드 생성
```

### 파이프라인 예시 데이터

```
고객사A: 제안 단계, 예상 금액 500만원, D+14일, 교육 분야
고객사B: 계약 단계, 예상 금액 1,200만원, D+3일, 피트니스
고객사C: 미팅 단계, 예상 금액 300만원, D+30일, 자격증
고객사D: 온보딩 중, 월 거래액 800만원, 성장률 +15%
고객사E: 이탈 위험, 월 거래액 200만원, 성장률 -20%, 최근 문의 3건
```

## 2. 설치 FAQ (프린트용)

| 증상 | OS | 해결법 |
|------|-----|--------|
| `permission denied` | Mac | `sudo curl -fsSL https://claude.ai/install.sh \| bash` |
| `claude` 명령어 인식 안됨 | Mac/Windows | 터미널 완전 종료 후 재시작 |
| 실행 정책 오류 | Windows | PowerShell에서 `Set-ExecutionPolicy RemoteSigned` 실행 |
| Node.js 없다는 에러 | 전체 | Claude Code는 Node.js 불필요 — 최신 버전은 자동 설치. 재설치 시도 |
| 로그인 화면 안 뜸 | 전체 | 브라우저 기본 설정 확인. `claude --login` 직접 실행 |
| "플랜 없음" 에러 | 전체 | Claude Team Standard 초대 메일 확인 → 수락 후 재시도 |
| npm 관련 에러 | Mac | `npm cache clean --force` 후 재설치 |
| 프록시/VPN 이슈 | 전체 | VPN 끄고 재시도 |

**비상 대응**: 설치가 도저히 안 되는 경우 → 앰버서더가 옆에서 직접 지원. 10분 이상 걸리면 다른 참가자 노트북으로 페어링 후 세션 계속 진행.

## 3. 앰버서더 브리핑

> 세션 1~2일 전에 DM 또는 짧은 통화로 전달

### 역할 요약

```
당신은 강사가 아닙니다. 함께 배우는 사람입니다.
다만, 비개발자 참가자가 기술적으로 막힐 때 옆에서 도와주세요.
```

### Session 1 중 구체적 역할

| 시간대 | 해야 할 일 |
|--------|-----------|
| 20:00~20:10 | 자기소개 때 앰버서더 역할 간단히 언급 |

### 주의사항

1. **참가자 대신 해주지 않기** — 힌트를 주고, 직접 입력하게 유도
2. **"이건 쉬운 건데"라고 말하지 않기** — 비개발자에게는 터미널 여는 것도 새로운 경험
3. **에러 메시지를 같이 읽기** — "이 에러 같이 볼까요?" 톤
4. **설치 10분 이상 막히면** → 진행자에게 알리기 (전체 진행에 영향)
5. **본인 자동화 프로젝트도 생각해오기** — Session 3부터 본인 것도 만듦
